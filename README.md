# lorumipsum

LoremIpsum:  A utility to provide a 'lorem ipsum' text for use in testing.

## Installation

Download from http://192.168.1.23:7070

## Usage

(-main): Provides lorem ipsum paragraphs

    $ java -jar lorumipsum-0.1.0-standalone.jar [args]
